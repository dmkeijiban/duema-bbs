# SNS Growth JP

Use this skill for Japanese SNS strategy and execution support.

## User Context

- User operates Duema BBS and related accounts.
- Main goal: increase stock-type revenue using AI, SNS, SEO, and community traffic.
- Existing assets include:
  - Duema BBS
  - X accounts such as `dmkeijiban` and `infurecard`
  - TikTok AI girl account `chmro55`
  - note monetization experience
- User prefers speed, practical experiments, and low manual work.

## Strategy Principles

- Do not chase impressions alone if follower conversion is zero.
- Use impressions as testing fuel, then convert with repeated account positioning.
- Avoid spammy URL-only posting, especially on new X accounts.
- Build a simple funnel:
  SNS hook -> engagement -> forum/thread/note -> follow/list/revenue.
- Favor small experiments with clear metrics over giant plans.

## X Growth Rules

For Duema accounts:

- Mix 70% native X posts, 20% soft forum references, 10% direct URL posts.
- Use replies and quote posts carefully; original posts usually build account identity better.
- Reuse viral formats, but rewrite with original angle and do not impersonate.
- Pin or repeat a clear reason to follow:
  - Price movement info
  - New-card reactions
  - Meta and deck advice
  - Forum discussion highlights

For new accounts:

- Warm up with native posts before heavy URL promotion.
- Avoid repeated identical hashtags or URLs.
- Prefer screenshots, short opinions, and questions before direct links.

## Giveaways

Giveaway campaigns can be used, but keep them clean:

- Ask for follow plus repost or reply; avoid too many conditions.
- Prefer replies when discussion volume matters.
- Prefer quotes when spread matters, but quality control is lower.
- Encourage screenshots of forum threads/comments as optional goodwill, not mandatory if it adds friction.
- State deadline, prize, eligibility, and winner selection clearly.

## TikTok / AI Girl Content

For AI girl content:

- Lean into consistency, character, and repeatable series.
- Use note/paid content as the monetization layer.
- Do not overcomplicate tracking if manual logging has no payoff.
- Repurpose winning hooks into X and note topics.

## Content Quality Bar

Do not output bland automated posts. Before suggesting copy, check:

- Would a real Japanese user post this?
- Does it sound specific to Duema or the account?
- Is there a hook in the first line?
- Is the CTA light enough for the platform?
- Does it build followers, not only clicks?

If generated copy is weak, say so and propose a better testing method instead of shipping it.

## Recommended Experiment Format

For each growth idea, provide:

- Goal
- Platform
- Post/campaign draft
- Why it may work
- What to measure
- Stop condition

Keep recommendations practical and short.
