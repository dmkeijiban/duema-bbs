# 2026-05-13 Auto summary TOP10

## Change

Auto-generated weekly and monthly summaries now collect TOP10 threads instead of TOP5.

## Reason

For SEO, TOP10 gives each summary page more useful content and more internal links while still staying focused.

## Updated

- Cron summary generation: `/api/summary/generate`

- Summary detail metadata text
- Admin summary helper text

## Notes

- Existing summaries keep their existing title and thread list.
- New weekly/monthly summaries will use TOP10.
