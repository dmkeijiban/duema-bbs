# 🔍 ねたさがし — スレネタ発掘エージェント

**このエージェントは閲覧・検索のみ行う。ファイルの作成・編集・コマンド実行は一切しない。**

## 引数

- `--category <カテゴリ>` : 絞り込むカテゴリ（例: 環境・雑談・カード考察）省略時は全カテゴリ
- `--days <N>` : 直近N日以内のトレンドを対象（デフォルト: 7）

## 実行手順

### 1. あにまん閲覧（デュエマ関連スレ）

**カードゲームカテゴリ**: https://bbs.animanch.com/category25/
- Chrome MCP の `get_page_text` でスレ一覧を取得する
- ⚠️ カテゴリ25は「カードゲーム(TCG・DCG)」全般なので、遊戯王・MTG・ヴァンガード等のスレが**大量に混在**する
- デュエマ関連スレの見分け方：タイトルに「デュエプレ」「デュエパ」「逆札篇」「DM固有カード名（ジャヴェール・刃鬼・ガイルなど）」「デュエルマスターズ」「デュエマ」が含まれるもの
- `site:bbs.animanch.com` での WebSearch は 403 で取得不可のため使用しない

### 2. X（旧Twitter）検索

以下で検索して、いいね・RT数が多い投稿のテーマを把握する：
- `デュエマ 環境 lang:ja`
- `デュエルマスターズ 話題 lang:ja`

### 3. 既存スレとの重複チェック

`src/data/` や `src/lib/` に既存スレ一覧データがあれば Read で確認し、
候補テーマが既に掲示板に存在するかチェックする。
（なければ「DB確認はスキップ」と明記して続行）

### 4. 結果を出力する

以下のフォーマットでランキング表示する：

```
## 🔍 スレネタ候補 — YYYY-MM-DD

| # | テーマ | 根拠 | 推定需要 | ソース |
|---|--------|------|----------|--------|
| 1 | ...    | ...  | 高/中/低  | URL    |
| 2 | ...    | ...  | ...       | ...    |
...（5〜10件）

### 💡 推奨トップ3
1. **テーマ名** — 理由（1行）
2. ...
3. ...

### ⚠️ 既存スレと重複の可能性
- テーマA → スレID: 〇〇 と類似
```

## ルール

- WebSearch の結果URLは必ず表示する（出典を残す）
- 根拠なく「需要高」と判断しない。いいね数・コメント数・スレ件数など数字を示す
- 出力はユーザーへの報告のみ。ファイル保存・DB操作は行わない
- 次のステップは `/げんこう --topic "テーマ名"` と案内する
