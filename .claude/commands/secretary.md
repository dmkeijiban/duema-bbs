# 🗂️ 秘書ロール

以下を順番に実行して、今日の状況を整理する。

## 実行手順

1. `ai-memory-vault` MCP → `memory.md` を読む
2. `ai-memory-vault` MCP → `projects.md` を読む  
3. `ai-memory-vault` MCP → 昨日の日付で `07-logs/daily/YYYY-MM-DD.md` を読む
4. `ai-memory-vault` MCP → 今日の日付で `briefing/YYYY-MM-DD.md` を読む（存在する場合）
5. 下記フォーマットで整理・表示する

## 出力フォーマット

```
## 📅 YYYY年MM月DD日（曜日）

### 🔥 今日やること
1. [30分以内] タスク名 — 理由
2. [1〜2時間] タスク名 — 理由
3. [時間があれば] タスク名

### ⚠️ 止まっているタスク
- タスク名 → ブロッカー：〇〇が必要

### 💡 今週の提案
- 短期で効果が出そうなこと（1〜2件）

### 📊 Duema BBS 状態
- 本番デプロイ：✅ or ⚠️
- 未公開コンテンツ：あり/なし
- 未実行マイグレーション：あり/なし
```

## ルール

- briefing ファイルが存在しない場合は、このセッションで内容を生成してから
  `ai-memory-vault` MCP で `briefing/YYYY-MM-DD.md` に保存する
- 完了したタスクは「今日やること」に含めない
- 優先順位は「今日しかできない > 積み上がっているリスク > 成長につながる」の順
