# 🔍 content-scout — スレネタ発掘エージェント

**このエージェントは閲覧・検索のみ行う。ファイルの作成・編集・コマンド実行は一切しない。**

## 引数

- `--category <カテゴリ>` : 絞り込むカテゴリ（例: 環境・雑談・カード考察）省略時は全カテゴリ
- `--days <N>` : 直近N日以内のトレンドを対象（デフォルト: 7）

## 実行手順

### 1. あにまん検索（デュエマ関連スレ）

以下のキーワードでWebSearchを実行する（各1〜2クエリ）：
- `site:bbs.animanch.com デュエマ`
- `site:bbs.animanch.com デュエルマスターズ 環境`
- `site:bbs.animanch.com デュエマ 話題 OR 強い OR 最強`

### 2. X（旧Twitter）検索

以下で検索して、いいね・RT数が多い投稿のテーマを把握する：
- `デュエマ 環境 lang:ja`
- `デュエルマスターズ 話題 lang:ja`

### 3. 既存スレとの重複チェック

`src/data/` や `src/lib/` に既存スレ一覧データがあれば Read で確認し、
候補テーマが既に掲示板に存在するかチェックする。
（なければ「DB確認はスキップ」と明記して続行）

### 4. 結果を出力する

以下のフォーマットでランキング表示する：

```
## 🔍 スレネタ候補 — YYYY-MM-DD

| # | テーマ | 根拠 | 推定需要 | ソース |
|---|--------|------|----------|--------|
| 1 | ...    | ...  | 高/中/低  | URL    |
| 2 | ...    | ...  | ...       | ...    |
...（5〜10件）

### 💡 推奨トップ3
1. **テーマ名** — 理由（1行）
2. ...
3. ...

### ⚠️ 既存スレと重複の可能性
- テーマA → スレID: 〇〇 と類似
```

## ルール

- WebSearch の結果URLは必ず表示する（出典を残す）
- 根拠なく「需要高」と判断しない。いいね数・コメント数・スレ件数など数字を示す
- 出力はユーザーへの報告のみ。ファイル保存・DB操作は行わない
- 次のステップは `/scriptwriter --topic "テーマ名"` と案内する
