## 1. 手直しが必要になる10の原因（分類）

実際に公開されたポストと、テンプレ生成の差分を観察して抽出した「ユーザーが直さざるを得なくなるパターン」。

| # | 原因 | 症状 | 直され方 |
|---|---|---|---|
| 1 | **変な改行** | 意味の切れ目でない場所で改行。1行が長すぎ/短すぎ | 「主語＋情景」で1行、「問い」で1行に再配置される |
| 2 | **テーマが弱い** | 誰でも知ってるカード、当たり障りないお題 | もっと尖った・世代が割れるカード名に差し替え |
| 3 | **デュエマ味が薄い** | 一般TCGでも成立する文。固有名詞が無い | 具体カード名・パック名・年号が足される |
| 4 | **説明しすぎ** | 前提・背景・補足が本文に入る | 説明文が丸ごと削除され、問いだけが残る |
| 5 | **抽象的すぎ** | 「強かったカード」など範囲が広すぎて答えにくい | 「あのころ一番強かった**友達のデッキ**」のように具体化 |
| 6 | **答えにくい** | リプで何を書けばいいか一瞬迷う | カード名1個で答えられる形に絞られる |
| 7 | **懐古が弱い** | 「今」の話になっている | 「あのころ」「小学生の頃」「当時」など時間軸が足される |
| 8 | **普段の言い回しと違う** | 丁寧すぎ/AIっぽい敬語。一人称が無い | 「ワイ」「僕」「〜なんでなん？？」「〜多くない？ｗ」に砕けて、**一人称の体験談が足される** |
| 9 | **1投稿に要素を詰めすぎ** | お題＋補足＋ハッシュタグ＋CTAが全部入り | 1投稿1メッセージに削られる |
| 10 | **スクロール中に一瞬で理解できない** | 3秒で意味が取れない。文頭に余計な語 | 結論・お題を最初の2行に前出しされる |

### 最頻出は #8（口調）と #4（説明しすぎ）
公開済みポストを見ると、テンプレ通りのきれいな文より、**一人称の砕けた口調＋体験談の一言**が付いたものが多い。Claudeはここを足せていないことが多いので、生成時に最優先で意識する。

## 2. before / after 例（既存データから・改変なし）

> いずれも既存の公開済みポストを「観察」したもの。投稿は一切変更していない。

### 例A：口調と一人称（原因#8）
**❌ テンプレ素のまま（Claudeが出しがちな形）**

→ ハッシュタグ無し / 一人称の素朴なツッコミ / 3行で即理解。

### 例B：体験談の自己開示（原因#8・#7）
**❌ 説明的**

```
**✅ 完成形に近い形**

```
→ お題（2行）＋ 自分から先に晒す一言（体験談）。これが入るとリプが倍増する（X-ideas.md 告白型の知見と一致）。

### 例C：説明しすぎの削除（原因#4・#10）
**❌ 詰め込み**
```
デュエマには色々な思い出があると思います。

```

```

「これ絶対誰も知らんだろ」
って思うカードを教えてください👇

#デュエマ
#デュエマ好きと繋がりたい
```
→ 前置き全削除。「老人会」の一語で文脈を立て、お題は一文。

## 3. 文体ルール（生成時に必ず適用）

### 3-1. 口調
- **一人称は「ワイ」または「僕」**。敬語に寄せすぎない。
- 砕けた語尾を使う：`〜なんでなん？？` / `〜多くない？ｗ` / `〜説` / `〜わかる人🙋`
- 笑いは全角小文字 `ｗ`（`w` や `（笑）` ではない）。
- **AIっぽい丁寧説明（「〜だと思います」「〜してみてください」）は禁止**。

### 3-2. 自己開示の一言（強力）
- お題系（優勝🏆・選手権・老人会）には、**自分から先に体験談を1行晒す**と参加率が上がる。
  - 例：`僕はアッシュ！なぜかフレームに入れて飾ってるｗ` / `ワイはエグゼズ・ワイバーンが返ってきてないｗ`
- 「みんな教えて」だけで終わらせない。

### 3-3. 改行
- **1行＝1意味**。「お題」と「問いかけ」は必ず別行。
- 1行は概ね全角20字以内。長い前提は行に分けず**削る**。
- お題部分は `「」` で囲って独立行にする（例：`デュエマの\n「一番ボコボコにされたデッキ」`）。

### 3-4. 文字数・構成
- 本文は概ね **2〜5行 / 全角60字以内**を目安（結果発表など一覧型は除く）。
- 構成は **［文脈ラベル or お題］→［問い］→（任意）［自己開示］** の順。
- スクロール中に**最初の2行で意味が伝わる**こと。前置きを置かない。

### 3-5. 「リプ」表記（「コメント」禁止）
- リプライは必ず **「リプ」**。`コメントで教えてください` は使わない → `リプで教えてください👇` / `気軽にリプ下さい！`
- ⚠️ 既存の公開済み・予約投稿には `コメント` 表記が混在している（6/23ルール確定前のもの）。**既存は直さない**。新規生成分のみ「リプ」で統一する。

### 3-6. 「の」補完ルール（既存ルールの再掲）
`デュエマ「○○」` → `デュエマの「○○」` に補う。ただし例外は補正しない：
- `デュエマで「○○」`（「で」は動詞的用法）
- `デュエマ老人会` `デュエマ黒歴史選手権`（固有複合語）
- `デュエマを一言で言うと？`（助詞「を」が先）
- ユーザーが意図的に選んだ形（`デュエマ「あのとき売っとけばよかった」`等）

### 3-7. 絶対禁止（全スロット・全時間帯）
- **URLを本文に入れない**（掲示板リンクはリプに入れる）。
- **掲示板の宣伝・誘導を本文に入れない**。
- ハッシュタグの詰め込み（多くて2個まで。19時・22時は基本無し）。

## 4. ハッシュタグ運用
- 7:00 / 12:00 老人会系：`#デュエマ` `#デュエマ好きと繋がりたい` まで（最大2個）。
- 19:00（物議・チャレンジ）：原則ハッシュタグ無し（砕けた問いの純度を保つ）。
- 22:00（選手権）：ハッシュタグ無し or 最小限。URL・掲示板誘導は禁止。
- 「多すぎると逆効果説」を採用。月・木のみハッシュタグ強めも可（X運用.md 仮説）。

## 4.5. スロット別ジャンル優先（2026-06-25 時間帯別分析）

> どの型をどのスロットに置くかの優先順位。詳細な運用ルールは [`x-posting-pdca-rules.md`](./x-posting-pdca-rules.md) §1 を参照。**固定企画を「毎日同じ型」で回さない**のが基本方針。

- **7:00**：**違和感/なんで系・軽い共感**を優先。優勝🏆を毎朝の固定にしない。朝は軽さで参加ハードルを下げる。
- **12:00**：**主砲枠**。VS型・世代判定・強い企画など、その週で最も尖った型を昼に置く。
- **19:00**：**世代判定・違和感/なんで系・チャレンジ枠（水曜）**を主役に。純議論（物議・「説」）は固定せずローテ1枠に留める。
- **22:00**：**選手権・結果発表**を中心に。懐古×自分ごと化（最高記録473Kインプの型）が夜に効く。
- **優勝🏆は週2〜3本まで**。リプは強いが単独インプは伸びにくいので、主役ではなく差し色で使う。
- **結果発表は22時が本命**。午前（11時前後）の小出しも候補。
- **展開は一気に全枠を入れ替えない**。どのスロットから寄せるかは固定せず、毎回の「予約案 → 手直し後」の差分を見ながら段階的に調整する（継続改善ループは [`x-posting-pdca-rules.md`](./x-posting-pdca-rules.md) §4.5）。

## 6. 重複防止ルール（被り検知）

> 同じテーマ・問い・切り口の使い回しは、フォロワーの飽き・インプ低下・「またこれか」離脱に直結する。**新規生成の前に必ず通す。**

### 6-1. 基本ルール
- 原則、**一度使った投稿テーマ・問い・切り口は再使用しない**。
- 似た内容を使う場合でも、**最低2〜3日は空ける**。
- **同じカード名・同じ問い方・同じ構文が連続しない**ようにする。
- 同じテーマを再利用するときは、**必ずニュアンスを変える**。
  - ✅ OK：「好きなカード」→「今でも売れないカード」→「子どもの頃の切り札」（対象・感情・時間軸がずれている）
  - ❌ NG：「一番好きなカード」→翌日「好きなカード教えて」（実質同じ問い）
- **優勝🏆・選手権・老人会・物議・違和感**など、同じ型でも**中身の問いが近すぎる**ものは避ける（型が同じでもお題は離す）。

### 6-2. 生成前の確認手順（必須）
1. **新規生成の前に、直近7日分の公開済み投稿・予約投稿を確認する**（`x-posting-pdca-rules.md` のデータソース早見表でTypefullyから取得）。
2. **直近2〜3日以内に近いテーマがある場合は、別テーマに差し替える**。
3. どうしても近いテーマを使う場合は、**対象・感情・時間軸・答え方のうち最低1つを変える**。
   - 対象：カード → デッキ → 環境 → プレイヤー像
   - 感情：好き → 後悔 → 懐かしい → 悔しい
   - 時間軸：今 → あのころ → 引退時 → 初めて買った日
   - 答え方：カード名1個 → VS二択 → 🙋挙手 → 数字・金額

### 6-3. 差し替え判定の早見
| 状況 | 判定 |
|---|---|
| 直近7日に同じ問い・同じ切り口がある | **差し替え**（別テーマへ） |
| 同テーマだが対象/感情/時間軸/答え方のどれかが違う | OK（2〜3日空ければ可） |
| 同じカード名が直近の投稿に連続で出ている | カード名を変える or テーマごと差し替え |
| 同じ型（選手権など）が間隔キャップ内（§3）かつ問いも近い | **差し替え** |
